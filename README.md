#### How to Build

现在的版本必须使用android studio 0.9+或者使用gradle 插件于0.14.+(或者自行修改gradle配置使用其他版本)

Android5.0版本要求必须使用JDK7+,因此我们的配置是使用target5.0,jdk7.android studio 可以在project配置中配置，其中必须选择jdk的目录是jdk7+的。
